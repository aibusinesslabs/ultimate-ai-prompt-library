### P-01368 - Comprehensive Technical User Guide Creation

**Category:** Writing & Content Creation  
**Subcategory:** Technical Writing  
**Difficulty:** Intermediate  
**Tags:** `technical writing` `user guide` `documentation` `software`

#### Prompt:

**Task:**  
Create a clear, concise, and comprehensive technical user guide for a software application that effectively instructs end-users on installation, configuration, and basic troubleshooting.

**Context:**  
You are a professional technical writer tasked with producing user-friendly documentation aimed at non-expert users of a new software product. The guide should enable users to independently set up and operate the software while minimizing support requests.

**Input:**  
- **Software Details**: {software_name}, {version} (name and version of the software)  
- **Installation Steps**: {installation_instructions} (step-by-step installation process)  
- **Configuration Options**: {configuration_settings} (key configurable parameters and their descriptions)  
- **Troubleshooting Tips**: {common_issues_and_solutions} (frequently encountered problems with solutions)

**Instructions:**  
1. Introduce the software by providing an overview and its primary purpose.  
2. Detail the installation process in a clear, stepwise manner, including prerequisites.  
3. Explain essential configuration settings, describing their function and impact.  
4. Provide a troubleshooting section addressing common issues with actionable solutions.  
5. Use clear headings, bullet points, and numbered lists to enhance readability and navigation.

**Constraints:**  
- Length: 300-500 words  
- Format: Use professional technical writing conventions with headings and lists  
- Style: Clear, concise, and accessible language targeted at non-technical users  
- Structure: Logical flow from introduction to installation, configuration, and troubleshooting

**Output Format:**  
A structured, professionally formatted technical user guide text with distinct sections titled: Introduction, Installation, Configuration, and Troubleshooting.

#### Use Case:  
This prompt is ideal for technical writers or product teams needing to generate user manuals or quick-start guides that facilitate user onboarding and reduce customer support workload. It suits software products targeted at a broad, non-expert audience requiring straightforward, actionable documentation.
