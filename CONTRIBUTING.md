# Contributing to the Ultimate AI Prompt Library

We welcome contributions from the community! Whether you want to add new prompts, suggest new categories, or fix a typo, your help is appreciated.

## How to Contribute

1.  **Fork the repository.**
2.  **Create a new branch** for your changes (`git checkout -b feature/your-feature-name`).
3.  **Make your changes.** Please adhere to the existing prompt format and quality standards.
4.  **Commit your changes** (`git commit -m 'Add some feature'`).
5.  **Push to the branch** (`git push origin feature/your-feature-name`).
6.  **Open a pull request.**

## Quality Standards

-   **Follow the Google Prompt Format:** Ensure all new prompts use the structured format (Task, Context, Input, etc.).
-   **Be Clear and Concise:** Prompts should be easy to understand and use.
-   **Provide Value:** Each prompt should serve a practical, real-world purpose.
-   **Check for Duplicates:** Before adding a new prompt, please check if a similar one already exists.

Thank you for helping make this the best AI prompt library available!
