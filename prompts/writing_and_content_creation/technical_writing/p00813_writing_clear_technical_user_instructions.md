### P-00813 - Writing Clear Technical User Instructions

**Category:** Writing & Content Creation  
**Subcategory:** Technical Writing  
**Difficulty:** Beginner  
**Tags:** `technical writing` `user instructions` `clarity` `documentation`

#### Prompt:

**Task:**  
Create clear and concise technical user instructions for a product or software feature that guide end-users through a specific process.

**Context:**  
You are a technical writer tasked with producing user-friendly instructions that simplify complex technical steps for non-expert users. The goal is to enhance usability and reduce support requests by providing well-structured guidance.

**Input:**  
- **Product Details:** {product_name}, {feature_name} (the product and specific feature to describe)  
- **User Scenario:** {user_goal} (the main objective the user wants to achieve)  
- **Step Details:** {step1}, {step2}, {step3}, {step4} (the key actions required to complete the process)  
- **Additional Notes:** {warnings}, {tips} (important cautions or helpful hints to include)

**Instructions:**  
1. Begin with a brief introduction that summarizes the purpose of the instructions and the user goal.  
2. Write each step clearly and sequentially, using simple language suitable for beginners.  
3. Incorporate any warnings or tips at relevant points within the steps to enhance user safety and success.  
4. Use bullet points or numbered lists to organize the instructions for easy scanning.  
5. Conclude with a short summary or next steps if applicable.

**Constraints:**  
- Length: 150-300 words  
- Format: Use numbered step-by-step instructions with clear headings  
- Style: Professional, clear, and accessible; avoid jargon  
- Structure: Introduction, stepwise instructions, conclusion

**Output Format:**  
A well-formatted technical user instruction document comprising an introductory paragraph, a numbered list of steps with integrated warnings and tips, and a closing summary.

#### Use Case:  
Use this prompt when creating beginner-friendly user manuals, help guides, or online support content. It benefits technical writers, product teams, and customer support specialists aiming to improve user experience and reduce confusion around product features.
